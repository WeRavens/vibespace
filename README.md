# VibeSpace

VibeSpace is a social app focused on fast expression, mood-based posting, and mobile-first interaction.

## Run Locally

**Prerequisites:** Node.js


1. Install dependencies:
   `npm install`
2. Copy environment values if needed from `.env.example`
3. Run the app:
   `npm run dev`
